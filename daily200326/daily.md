# 📅 Dia 20 - Dataplex + BigQuery

## 📚 Estudo
Neste dia estudei os conceitos do Dataplex, entendendo seu papel na organização e governança de dados dentro de um data lake. Aprendi como ele permite centralizar diferentes fontes de dados, estruturadas e não estruturadas, facilitando o gerenciamento e a catalogação.

Também explorei como o Dataplex se integra com outros serviços do Google Cloud, principalmente o BigQuery, permitindo uma análise mais eficiente dos dados organizados.

## ⚙️ Prática
Realizei uma aplicação prática no BigQuery utilizando o Cloud Console, executando consultas SQL e explorando datasets disponíveis. Consegui visualizar como os dados podem ser organizados e consultados dentro de um ambiente controlado.

## ✅ Aprendizado
Compreendi a importância da governança de dados e como o Dataplex ajuda a manter organização, qualidade e controle em ambientes de dados complexos.
