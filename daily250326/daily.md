# Dia 25 - Dataform

## Estudo
Estudei o Dataform e sua função na criação e organização de pipelines de dados utilizando SQL. Entendi como ele permite estruturar transformações de dados de forma mais organizada e versionada.

Também explorei como o Dataform se integra ao BigQuery, permitindo gerenciar tabelas, views e transformações de forma automatizada.

## Prática
Utilizei o Dataform no Cloud Console para criar e executar transformações SQL, organizando etapas de processamento de dados dentro de um fluxo lógico.

## Aprendizado
Compreendi como o Dataform ajuda na organização e manutenção de pipelines, facilitando o trabalho com grandes volumes de dados e múltiplas transformações.
