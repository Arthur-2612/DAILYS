# Dia 24 - Dataflow

## Estudo
Neste dia foquei no Dataflow, aprendendo sobre processamento de dados em batch e streaming. Entendi como ele utiliza o Apache Beam para construir pipelines de dados escaláveis.

Também estudei como o Dataflow pode ser utilizado para processar grandes volumes de dados em tempo real ou sob demanda, com integração direta ao BigQuery.

## Prática
Realizei atividades práticas no lab do Google Cloud, executando pipelines e observando como os dados são processados e enviados para o BigQuery.

## Aprendizado
Aprendi como construir pipelines eficientes e como o Dataflow facilita o processamento distribuído de dados em larga escala.
