# Dia 23 - Dataprep

## Estudo
Estudei o Dataprep e seu papel no processo de preparação de dados. Aprendi como ele é utilizado para limpar, transformar e organizar dados antes de serem utilizados em análises.

Entendi conceitos importantes como tratamento de valores nulos, padronização de dados e transformação de formatos, que são essenciais em qualquer pipeline de dados.

## Prática
Utilizei o Dataprep no Cloud Console para aplicar transformações em um conjunto de dados, realizando limpezas e ajustes necessários para deixá-los prontos para análise.

## Aprendizado
Percebi como a preparação de dados é uma etapa fundamental e como ferramentas como o Dataprep facilitam esse processo de forma visual e eficiente.
