# Dia 26 - Dataproc + Spark

## Estudo
Neste dia estudei o Dataproc e sua utilização para processamento distribuído com Apache Spark. Aprendi como clusters são criados e utilizados para executar tarefas de grande escala.

Também entendi melhor o funcionamento do Spark, incluindo conceitos como processamento paralelo e manipulação de grandes volumes de dados.

## Prática
Realizei uma aplicação prática no Google Cloud Console utilizando o Dataproc, executando jobs com Spark e analisando os resultados obtidos.

## Aprendizado
Aprendi como utilizar o Dataproc para executar aplicações distribuídas e como o Spark pode ser utilizado para processamento eficiente de dados.
